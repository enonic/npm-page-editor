# Step 11.3 — Dependency pruning

> Parent: [11-INIT.md](11-INIT.md)

## Goal

Drop every runtime and type dependency that only the legacy codebase used. Evaluate the two `file:` dependencies on `.xp/dev/lib-admin-ui` and `.xp/dev/lib-contentstudio`.

**Blocker:** 11.2 must have landed — deleting `page-editor/` and `new-ui/` is what makes these dependencies unused.

## Scope

```
UPDATE package.json          ← remove unused deps
UPDATE tsconfig.app.json     ← remove unused type packages, drop .xp/dev/* includes
UPDATE pnpm-lock.yaml        ← regenerated by pnpm install
```

## package.json

**`dependencies` to remove:**
- `jquery`, `jquery-ui`, `jquery-simulate`
- `mousetrap`
- `q`
- `dompurify`
- `nanoid` — verify v2 usage first; keep only if v2 relies on it.

**`devDependencies` to remove:**
- `@types/jquery`, `@types/jqueryui`, `@types/mousetrap`, `@types/q`, `@types/ckeditor`
- `@rollup/plugin-inject`

**Evaluate (remove if no v2 imports remain):**
- `@enonic/lib-admin-ui` (`file:.xp/dev/lib-admin-ui`)
- `@enonic/lib-contentstudio` (`file:.xp/dev/lib-contentstudio`)

Confirm with `grep -r "@enonic/lib-" src/main/resources/assets/js/v2/` returning zero matches before dropping.

## tsconfig.app.json

- Remove from `compilerOptions.types`: `jquery`, `jqueryui`, `q`, `ckeditor`, `mousetrap`.
- Remove from `include`: `.xp/dev/lib-admin-ui/**/*.ts` and `.xp/dev/lib-contentstudio/**/*.ts` (only if the `file:` deps were dropped).

## Dependencies

Requires **11.2** (legacy directory deletion).

## Verification

- `pnpm install` runs clean; lockfile regenerates without errors.
- `pnpm check` green.
- `pnpm build:dev` and `pnpm build:prod` succeed.
- `pnpm build-storybook` succeeds.
- `node_modules/jquery` absent after install (sanity check).
- `grep -rE "require\\(|from ['\"]jquery" src/main/resources/assets/js/` returns nothing.
