# Step 11.6 — Storybook and final polish

> Parent: [11-INIT.md](11-INIT.md)

## Goal

Final cleanup pass. Remove the jQuery global bootstrap from `.storybook/preview.tsx`, audit every story for lingering legacy imports, and run the full build matrix one last time to verify the v2 migration is complete.

## Scope

```
UPDATE .storybook/preview.tsx
UPDATE any story referencing deleted modules
```

## .storybook/preview.tsx

- Remove the jQuery global bootstrap (`globalThis.$ = jQuery`, `globalThis.jQuery = jQuery`) — nothing consumes it after 11.3.
- Keep the dark-mode theme sync MutationObserver — it syncs Storybook's theme class with v2's `$theme` store and is already v2-native.
- Verify the `withThemeByClassName` decorator still works.

## Story audit

- `grep -rE "from ['\"].*page-editor/" .storybook/ src/` — must return zero.
- `grep -rE "from ['\"].*new-ui/" .storybook/ src/` — must return zero.
- For each hit, either update to the v2 equivalent or delete the story if the subject no longer exists.

## Dependencies

Requires **11.5** (so no `preact/compat` imports linger in stories either).

## Verification

- `pnpm build-storybook` succeeds.
- `pnpm storybook` boots cleanly (smoke test, don't run autonomously).
- `pnpm check` green.
- `pnpm build:dev` and `pnpm build:prod` succeed.
- `./gradlew build -Penv=dev` succeeds — full Gradle build is the release gate.
- Final grep: `grep -rE "jquery|mousetrap|from ['\"]q['\"]|dompurify" src/ .storybook/` returns zero matches outside of comments/docs.
